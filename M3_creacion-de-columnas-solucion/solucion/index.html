Introducción al desarrollo web

¿Qué es el HTML?

El HTML es un lenguaje de marcado de hipertexto que se utiliza para elaborar las páginas web y está conformada por un grupo de etiquetas que muestran el contenido de diferente forma como: un enlace a otra página, una imagen, etc.

¿Qué es el CSS?

El CSS es una hoja de estilo en cascada enfocada en definir, crear y mejorar la presentación de un documento HTML y está conformada por un grupo de reglas que permiten dar estilos gráficos a los elementos.
